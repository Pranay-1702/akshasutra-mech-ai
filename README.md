# 🧠 Akshasutra — Mechanical Engineering AI Assistant

Akshasutra is an AI-powered chatbot trained on standard Mechanical Engineering textbooks.

## 🚀 Features
- RAG-based intelligent answers
- Supports SOM, Thermodynamics, FEM, Robotics
- Source-based responses (book + page)
- Gemini-powered LLM
- Clean chat UI

## 🔑 Usage
1. Enter your Gemini API Key
2. Ask any mechanical engineering question
3. Get accurate answers with sources

## ⚙️ Tech Stack
- LangChain
- FAISS
- Streamlit
- Google Gemini

## 🌐 Deployment
Deployed using Streamlit Cloud
